## Tentang Immigration Package

Paket ini merupakan paket imigrasi dengan konstruksi mongodb

### Versi 
0.1.0

### Fitur
- Autentikasi dengan email dan hashed password
- Otorisasi berdasarkan role untuk kantor tertentu (disebut visa)

### Struktur folder
- Infrastructures
	- Aggregates
	- Exceptions
	- Guid
	- Models
- Subdomains
	- Pengguna
- UbiquitousLibraries
